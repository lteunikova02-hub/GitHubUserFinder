# GitHub User Finder
**Автор:** Бекшоков Султан
**Дата сдачи:** 02.05.2026

## Описание программы

**GitHub User Finder** - приложение для поиска пользователей GitHub с возможностью добавления в избранное. Реализовано на Python с использованием tkinter и GitHub API.

## Функционал

* Поиск пользователей GitHub по имени.
* Отображение результатов в списке.
* Добавление пользователей в избранное.
* Сохранение избранного в JSON‑файл.
* Проверка корректности ввода.

## Требования для запуска

Что нужно установить на компьютер:

- Python 3.10 или выше
- Библиотеки: `pip install matplotlib requests`
---
## Как запустить
```bash
git clone https://github.com/lteunikova02-hub/GitHubUserFinder
cd githubuserfinder
python main.py
``` 
## Как пользоваться 

1.При запуске откроется окно приложения с полем ввода и кнопками.

2.Введите имя пользователя GitHub в поле «Введите имя пользователя GitHub:».

3.Нажмите кнопку **«Найти»** — в списке ниже отобразятся результаты поиска (логин, имя, количество публичных репозиториев).

4.Чтобы добавить пользователя в избранное:

* выберите его в списке результатов;

* нажмите кнопку **«Добавить в избранное»** — пользователь появится в списке избранных внизу окна.

5.Чтобы удалить пользователя из избранного:

* выберите его в списке избранных;

* нажмите кнопку **«Удалить из избранного»** — пользователь будет удалён из списка и сохранённых данных.

6.Избранные пользователи автоматически сохраняются в файле favorites.json после каждого изменения.

## Пример работы

**1.Поиск пользователя:**

В поле ввода пишем: octocat

Нажимаем **«Найти»**

В списке результатов появляется: octocat - The Octocat - 8 репозиториев

**2.Добавление в избранное:**

Выделяем строку с octocat в списке результатов

Нажимаем **«Добавить в избранное»**

Появляется сообщение: «octocat добавлен в избранное!»

Пользователь octocat появляется в списке «Избранные пользователи:» внизу окна

**3.Удаление из избранного:**

Выделяем octocat в списке избранных

Нажимаем **«Удалить из избранного»**

Появляется сообщение: «octocat удалён из избранного!»

Пользователь исчезает из списка избранных и удаляется из favorites.json

**4.Обработка ошибок:**

Если поле поиска пустое и нажата **«Найти»** → сообщение: «Поле поиска не должно быть пустым!»

Если пользователь не найден → сообщение: «Пользователь не найден (код: 404)»
